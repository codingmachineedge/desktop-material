Git, rebuilt in Material Design 3.

A Material Design 3 shell, live now

Everything below is implemented and on main — the full GitHub Desktop workflow, rebuilt around Material Design 3.

🎨

M3 Expressive shell

Animated light/dark theming, soft elevation, an app bar with an inline pill menu, and floating radius-24 workspace cards.

🧭

Icon navigation rail

A left rail for Changes (with a badge), History, Branches, Settings, and your account avatar — plus a floating pill toolbar with repo and branch chips and a sync pill.

🗂️

Browser-like repo tabs

Tabs bound per-account to repositories, with inline rename and per-tab title styling — bold, italic, size, colour, font, alignment.

👥

Real multi-account

Multiple identities per host, each with its own tabs, repos, and settings. Switching the active account switches the whole workspace context.

🕔

Versioned settings

Every settings and tab change auto-commits to a local git repo, with a non-modal history side sheet for diffs, undo, redo, and restore. Each action appends an audit commit instead of rewriting history.

🪟

Non-modal dialogs

Dialogs float without blocking the app, drag by their headers, cascade, and come to front on focus. Preferences is an MD3 940×660 dialog; repo and branch pickers are side sheets.

On the roadmap

Planned next

These are the queued milestones — not yet implemented. Progress is tracked in the project's plan.

🔔

Notification centre Roadmap

A bell and side panel backed by its own local git repo — unread badges, mark read/unread, and delete.

🔍

Regex-powered search Roadmap

Every search bar gets filter chips, a regex toggle, and a full builder — anchors, classes, quantifiers, groups, lookaround, live tester.

⧉

Multi-clone Roadmap

Checkbox-select many repos, filter by org chips, clone in parallel or one-by-one, and export/import repo lists as URLs.

⚡

Automation Roadmap

One-click commit & push with a Copilot-written message, scheduled auto commit/push and auto pull, and merge-all with Copilot conflict resolution.

▷

GitHub Actions panel Roadmap

Browse workflow runs, filter by status/branch/event, re-run or re-run-failed, inspect job steps, and read logs in-app.

🤖

Built-in MCP server Roadmap

Drive the app from AI agents over a bundled MCP server, with a local HTTP/CLI fallback.

🏢

Organizations & scaling Roadmap

Browse and clone full org repo lists, publish into an org, and scale the whole UI from 50–200% with auto-fit.

🦊

GitLab & parity Roadmap

Self-hosted GitLab (endpoint + PAT) and Bitbucket sign-in, plus desktop-plus parity: commit search, commit graph, stashes, pinning, pull-all.
